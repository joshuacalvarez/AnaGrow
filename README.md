# AnaGrow
## Word-Ladder puzzle mobile game made in Unity
### I'm a huge fan of daily puzzles and word games, so this personal project fulfilled that craving!
![Demo animation](Github/AnaGrowGif.gif)
- Dynamic UI elements for an engaging player experience
- Programmed logic for word validation, keyboard input, and real-time updates using event listeners
- Utilized JSON data handling and modular scripts to improve scalability and mantainability
